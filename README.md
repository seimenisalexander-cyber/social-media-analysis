# Social Media & Mental Health Analysis

## 📊 Project Overview
This project analyzes how social media usage impacts stress levels using real-world data.

## 📈 Key Insights
- The average daily social media usage is approximately 4.5 hours.
- Instagram is the most popular platform among users.
- Higher social media usage is associated with higher stress levels.

## 🛠️ Tools Used
- Excel
- Pivot Tables
- Data Visualization

## 📁 Dataset
Teen mental health dataset including:
- Social media usage
- Sleep hours
- Stress & anxiety levels

## 🎯 Objective
To explore patterns between social media behavior and mental health indicators.
